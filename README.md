# Export My Wallet

Export any Solana wallet to CSV in seconds.

## 🚀 What is this?

Export My Wallet is a simple tool that converts any Solana wallet’s transaction history into a clean, usable CSV file.

Supports exporting full Solana transaction history for wallets, including swaps, transfers, and token activity.

Useful for:
- Tax reporting
- Transaction tracking
- Wallet analysis
- Record keeping

## ⚡ Why I built this

Solana explorers are great for browsing, but not for exporting clean data.

Most tools are either:
- Overcomplicated
- Expensive
- Or slow

This was built to make the process fast and simple.
